Doggo Dash

A 2D arcade-style game where the player controls a dog character using the keyboard's arrow keys. The primary objective is to maximize the score by collecting treats while avoiding collisions with bomb objects. The game uses a finite state machine to manage play, pause, and game-over states, with real-time rendering, collision detection, and randomized object spawning. Scoring rewards collecting treats and penalizes missed treats or bomb collisions, showcasing core concepts of event-driven programming and interactive graphics. Created with HTML, CSS, and JavaScript.

Game of Life

Simulates a Game of Life board game, where players interact with a menu-driven interface to make choices that influence their progress. Gameplay consists of selecting actions, navigating events, and completing missions determined by the state of the game board. The implementation emphasizes control flow, state management, and user interaction through a structured menu system. Created with C++.
